### Prasyarat

Pastikan Anda sudah menginstal **Node.js** (direkomendasikan versi LTS) dan **npm** (atau **Yarn** / **pnpm**).

### 1. Kloning Repositori

Buka terminal Anda dan *clone* repositori dari GitHub:

```bash
git clone [https://github.com/MuhamadRaihanTaqwa/tokocimi.git](https://github.com/MuhamadRaihanTaqwa/tokocimi.git)
cd tokocimi

2. Instal Dependensi
Instal semua paket yang diperlukan (yang tercantum dalam package.json):
npm install 
# atau yarn install

3. Menjalankan Server Pengembangan
Jalankan proyek dalam mode pengembangan (development mode):
npm run dev
# atau yarn dev
Aplikasi akan berjalan di http://localhost:3000 (atau port lainnya).

4. Build untuk Produksi (Opsional)
Untuk membuat build siap produksi:
npm run build
npm start
